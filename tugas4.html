WEBSITE STATIS :
Website statis merupakan halaman web yang jarang atau bahkan tidak diperbaharui
oleh pemilik situs dalam kurun waktu tertentu Dapat dikatakan konten yangberada
pada website statis akan tetap sama sejak website tersebut dibuat.

Static websites are web pages that are rarely or even not updated
by the site owner within a certain period of time. It can be said
that the content that ison a static website will remain the same since the website was created

WEBSITE DINAMIS :
Sedangkan konten pada website dinamis dapat diperbaharui dalam kurun waktu tertentu.
Dalam website dinamis, pengelola atau pemilik website menyediakan ruang interaksi
antara pengelola dan pengguna website, bisa dalam bentuk kolom komentar, tombol
reaksi, dan tombol share.

While the content on dynamic websites can be updated within a certain time.
In dynamic websites, website managers or owners provide interaction space
between managers and website users, it can be in the form of comment fields
reaction buttons, and share buttons
